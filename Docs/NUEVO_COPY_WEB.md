# Nuevo copy web ASESPRO

Fecha: 2026-05-01

Objetivo: reemplazar el copy actual de la web por textos mas claros, locales y comerciales, alineados con el negocio real de ASESPRO.

## Criterio general de comunicacion

ASESPRO debe sonar como una empresa seria, profesional y actualizada, pero cercana. El foco principal de la web es inmobiliario: alquiler y venta de casas, apartamentos y terrenos en Paso de los Toros, Pueblo Centenario y alrededores.

El servicio de limpieza debe aparecer como una unidad de servicio profesional separada, disponible en Paso de los Toros, Montevideo y Maldonado, para empresas y particulares. Para este servicio, el llamado a la accion principal debe ser el formulario de contacto para solicitar presupuesto personalizado.

Evitar expresiones demasiado aspiracionales o poco reales para el mercado local, como "espacios de autor", "legado", "curaduria arquitectonica", "certificacion ISO" o "premium" cuando no haya una prueba concreta. Priorizar claridad, confianza, disponibilidad y contacto rapido.

# Notas de implementacion de copy

1. Cambiar el tono actual de lujo generico por un tono profesional, local y claro.
2. Corregir textos sin acentos cuando se actualicen los componentes.
3. En inmobiliaria, mantener WhatsApp como canal principal de conversion.
4. En limpieza, usar formulario como canal principal para presupuesto personalizado.
5. Reemplazar imagenes de limpieza que hoy usan fotos de propiedades por imagenes de limpieza profesional.
6. Revisar propiedades de ejemplo: no usar Durazno si el foco comercial es Paso de los Toros, Pueblo Centenario y alrededores.
